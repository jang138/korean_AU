** `./run_experiments.sh` file 에 대한 사용 가이드입니다.

1. start/status/stop/tail 서브커맨드를 제공하며, 예시 3개를 nohup으로 실행하고 각각 PID/로그를 관리합니다.
2. 로그는 korean_AU/logs/, PID는 korean_AU/nohup/pids/에 저장됩니다.
3. 사용법:
    - 시작: bash nohup/run_experiments.sh start
    - 상태: bash nohup/run_experiments.sh status
    - 종료: bash nohup/run_experiments.sh stop
    - 로그 보기: bash nohup/run_experiments.sh tail run_bert_lr5e-4
